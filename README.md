# Conoce Salta — Proyecto de Turismo (HTML • CSS • JS)

**Proyecto demo**: sitio estático que muestra lugares turísticos de la provincia de Salta.  
Este repositorio es un prototipo educativo — algunos datos son ficticios y sirven para probar diseño y funcionalidades.

---

## ✨ Qué incluye este proyecto
- Portada con hero + buscador.
- Listado de destinos destacados.
- Página de resultados con búsqueda filtrada (JSON local).
- Página de detalle por lugar.
- Sección FAQ (acordeón CSS).
- Sección de números de emergencia.
- Componentes accesibles y responsive (móvil/desktop).
- Archivos: HTML, CSS, JavaScript y `data/lugares.json` (datos de ejemplo).
